# Objective

Implement the shared layout components for Project Ever After.

---

# Context

Read before making changes:

* AGENTS.md
* docs/02-design-system.md
* docs/03-technical-architecture.md
* prompts/foundation/05-shared-components.md

Follow all applicable Cursor rules.

---

# Scope

Implement only the shared layout components.

---

# Files Allowed to Modify

Create files only inside:

components/shared/layout/

---

# Components

Create:

* Container
* Section
* MaxWidth
* Stack

Each component must:

* Be fully typed
* Accept className overrides
* Accept children where appropriate
* Be responsive
* Follow the Design System
* Use composition over configuration

---

# Constraints

Do NOT:

* Create typography components
* Create display components
* Create motion components
* Create feature-specific components
* Modify globals.css
* Modify app/layout.tsx

---

# Acceptance Criteria

* All four layout components are reusable.
* Components are responsive.
* Components are accessible.
* Components contain no business logic.

---

# Deliverables

Provide:

1. Files created
2. Purpose of each component
3. Any reusable patterns introduced

---

# Stop Condition

Stop after completing the layout components.

Wait for approval before continuing.
