# Objective

Implement the global navigation experience for Project Ever After.

---

# Context

Read:

* AGENTS.md
* docs/01-product-specification.md
* docs/02-design-system.md
* docs/03-technical-architecture.md

---

# Scope

Implement only the navigation.

---

# Requirements

Create a navigation that:

* Supports mobile and desktop.
* Reflects the story-driven flow.
* Is accessible via keyboard.
* Highlights the active section.
* Uses smooth scrolling where appropriate.
* Matches the project's visual identity.

Expected sections include:

* Invitation
* Our Story
* Timeline
* Proposal
* Wedding Day
* FAQ
* RSVP

---

# Constraints

* Do not implement page content.
* Do not implement RSVP logic.
* Keep the navigation reusable.

---

# Acceptance Criteria

* Navigation is responsive.
* Navigation is accessible.
* Navigation matches the Design System.
* Navigation is ready for future sections.

---

# Stop Condition

Stop after the navigation component is complete.

Provide a summary of the implementation.
