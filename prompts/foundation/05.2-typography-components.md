# Objective

Implement the shared typography components for Project Ever After.

---

# Context

Read before making changes:

* AGENTS.md
* docs/02-design-system.md
* docs/03-technical-architecture.md
* prompts/foundation/05-shared-components.md

Follow all applicable Cursor rules.

---

# Scope

Implement only the typography component library.

---

# Files Allowed to Modify

Create files only inside:

components/shared/typography/

---

# Components

Create:

* PageTitle
* SectionTitle
* Eyebrow
* Lead

Each component must:

* Use the established typography tokens
* Be fully typed
* Accept className overrides
* Accept children
* Remain content-agnostic

Do not hardcode wedding-specific text.

---

# Constraints

Do NOT:

* Create layout components
* Create display components
* Create motion components
* Create feature components

---

# Acceptance Criteria

* Typography hierarchy is reusable.
* Components are responsive.
* Components are accessible.
* No duplicated typography patterns exist.

---

# Deliverables

Provide:

1. Components created
2. Props exposed
3. Reusability notes

---

# Stop Condition

Stop after completing the typography components.

Wait for approval before continuing.
