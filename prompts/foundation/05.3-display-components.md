# Objective

Implement the shared display and motion foundations for Project Ever After.

---

# Context

Read before making changes:

* AGENTS.md
* docs/02-design-system.md
* docs/03-technical-architecture.md
* prompts/foundation/05-shared-components.md

Follow all applicable Cursor rules.

---

# Scope

Implement only the shared display and motion components.

---

# Files Allowed to Modify

Create files only inside:

components/shared/display/
components/shared/motion/

---

# Display Components

Create:

* Divider
* DecorativeFlourish
* LoadingScreen

---

# Motion Components

Create:

* FadeIn
* Reveal
* Stagger

Requirements:

* Respect reduced-motion preferences.
* Be reusable wrappers.
* Accept className where appropriate.
* Remain independent of page content.

---

# Constraints

Do NOT:

* Implement envelope animations
* Create navigation
* Create page components
* Create feature-specific UI

---

# Acceptance Criteria

* Display components are reusable.
* Motion components are reusable.
* Components are accessible.
* Components follow the Design System.

---

# Deliverables

Provide:

1. Files created
2. Purpose of each component
3. Any reusable animation patterns introduced

---

# Stop Condition

Stop after completing the display and motion components.

Wait for approval before continuing.
