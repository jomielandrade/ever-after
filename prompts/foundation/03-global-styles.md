# Objective

Implement the global application styles for Project Ever After.

---

# Context

Read before making changes:

* AGENTS.md
* docs/02-design-system.md

Follow all applicable Cursor rules.

---

# Scope

Implement only the global styles.

---

# Files Allowed to Modify

* app/globals.css

---

# Files That Must Not Be Modified

* app/layout.tsx
* components/
* features/

---

# Requirements

Configure:

* Body defaults
* Typography defaults
* Smooth scrolling
* Font smoothing
* Text selection styling
* Link styling
* Default focus styling
* Base heading styles
* Base paragraph styles

Use the design tokens defined in the theme.

Do not hardcode colors that already exist as CSS variables.

---

# Constraints

* Do not create components.
* Do not modify metadata.
* Do not create pages.
* Do not introduce custom utility classes unless necessary.

---

# Acceptance Criteria

* Global styles are consistent.
* Typography follows the Design System.
* The application has sensible default styling.
* No feature-specific styling is added.

---

# Stop Condition

Stop after global styles are complete.

Summarize:

* Files modified
* Styling decisions
* Any recommendations
