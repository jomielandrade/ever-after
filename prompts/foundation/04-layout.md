# Objective

Create the application's root layout foundation.

---

# Context

Read:

* AGENTS.md
* docs/02-design-system.md
* docs/03-technical-architecture.md

---

# Scope

Implement only the application layout.

---

# Files Allowed to Modify

* app/layout.tsx

---

# Requirements

Configure:

* Root HTML structure
* Language attribute
* Body layout
* Theme classes
* Metadata organization
* Font variables
* Global providers (if required)

The layout should be minimal and reusable.

---

# Constraints

* Do not implement navigation.
* Do not create pages.
* Do not create feature components.

---

# Acceptance Criteria

* Root layout follows Next.js 16 App Router best practices.
* Fonts are applied.
* Global styles are loaded.
* Metadata is organized.

---

# Stop Condition

Stop after the layout foundation is complete.

Provide a concise implementation summary.
